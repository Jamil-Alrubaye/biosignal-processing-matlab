# Lab 5 – SEP Analysis

This laboratory exercise focuses on SomatoSensory Evoked Potential (SSEP) signal analysis using MATLAB.

## Tasks Completed

### Task 1 – Raw Signals and Ensemble Average
- Loaded 100 SSEP trials
- Converted signal amplitude to µV
- Plotted all raw trials
- Calculated ensemble average

### Task 2 – SNR Estimation
- Estimated noise variance
- Calculated signal energy
- Computed SNR in dB
- Estimated SNR improvement after averaging

### Task 3 – Chirp Modeling
- Applied linear chirp model fitting using PSO toolbox
- Compared fitted chirp with averaged SSEP signal

### Task 4 – Feature Extraction
- Extracted chirp delay
- First and second peak amplitudes
- Peak locations (delays)
- Estimated third peak location
